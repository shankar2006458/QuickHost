# QuickHost 🚀

Free and modern file hosting web app built with HTML, Tailwind CSS, JavaScript, and Supabase.

Upload any file instantly and get a shareable direct link in seconds.

---

## ✨ Features

* ⚡ Fast drag & drop uploads
* 🔗 Instant shareable links
* 📋 Copy Direct / HTML / Markdown links
* 🖼️ Image preview support
* 🎬 Video preview support
* 🎵 Audio preview support
* 📄 PDF & archive file support
* 📁 Recent upload history
* 🪟 Smooth animated history popup
* 📱 Fully responsive UI
* 🌌 Modern glassmorphism design
* ☁️ Supabase Storage integration
* 🆓 No signup required

---

## 🛠️ Tech Stack

* HTML5
* Tailwind CSS
* Vanilla JavaScript
* Supabase Storage


---


## 📁 Supported File Types

* Images
* Videos
* Audio
* PDFs
* ZIP / RAR
* Any file type
